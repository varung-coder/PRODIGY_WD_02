## 📖 Overview

This project is a sophisticated, interactive Stopwatch Web Application designed to provide highly accurate time tracking. Developed as part of a web development internship, it combines robust functionality—such as start, pause, resume, reset, and lap recording—with a stunning, modern glassmorphism aesthetic. The clean user interface and smooth micro-animations deliver a premium, SaaS-like user experience across all devices.

<br />

## ✨ Key Features

- **Precision Time Tracking:** Highly accurate time measurement displayed in `HH:MM:SS.ms` format.
- **Dynamic Lap Recording:** Easily record and list multiple lap times, calculating the difference between laps alongside total elapsed time.
- **Glassmorphism UI:** Features a sleek, frosted glass effect with soft shadows over a futuristic, animated gradient background.
- **Interactive Animations:** Smooth button transitions, hover glow effects, and a floating card layout for a highly engaging user experience.
- **Fully Responsive:** Perfectly adapts its layout to look beautiful on mobile phones, tablets, and desktop monitors.
- **Zero Dependencies:** Pure vanilla JavaScript implementation with no external libraries.

<br />

## 🛠 Tech Stack

- **HTML5:** Semantic structure and accessible DOM elements.
- **CSS3:** Advanced styling including CSS variables, Flexbox, backdrop-filters for glassmorphism, and custom keyframe animations.
- **JavaScript (ES6+):** Core logic handling intervals, state management, and dynamic DOM updates.
- **Font Awesome:** Scalable vector icons for intuitive user controls.
- **Google Fonts:** Utilizing the `Inter` font family for modern, clean typography.
